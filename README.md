# 주간 업무 리포트 생성기

## 개요
- 아이디어 번호: 20
- 한 줄 컨셉: 작업 로그를 주간 보고서로 자동 작성
- 수익모델: 구독
- MVP 난이도: 하
- 카테고리: 생산성

## 기술 스택
- Flutter (Android/iOS/Web)
- Supabase (Postgres + Auth + RLS)

## 실행
```bash
flutter pub get
flutter analyze
flutter test
flutter run
```

## 백엔드
- schema: `backend/supabase/schema.sql`
